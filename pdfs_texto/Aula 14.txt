--- PAGE 3 ---
● Visão
– É um conjunto resultado de uma consulta
– Tabela derivada a partir das tabelas do BD
– Pode ser vista coo uma tabela virtual, o usuário manipula como se 
fosse uma tabela normal
● Visões fazem parte do esquema externo de um Banco de Dados
● CREATE VIEW nome AS
SELECT atributos FROM tabela [JOIN tabela] [WHERE 
filtros]

Como criar um índice

andar

10)Buscar o CPF e o nome de todos os médicos e, para aqueles médicos com consultas marcadas,
